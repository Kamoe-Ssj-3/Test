Nieuwe map aanmaken--> in die map cmd openen en "git clone /link"

In die map cmd en "git status"

Dan "README.md" toevoegen en weer "git status" runnen in cmd

Dan om toe te voegen git add "file". Staging area

"git commit -m "Message" ". Local repository

"git push". Remote repository

"git fetch". To see updates in the git directory

Branch:
git fetch, then checkout the branch --> git checkout <branch_name>

git checkout -b <branch_name>
git push -u origin <branch_name>
git branch om te kijken op welke branch je zit

git merge <branch_you_want_to_merge> (to check what changes, run `git diff`)
On Github sit, merge branch by creating a pull request
